Detailed Description

The json_variant class provides an interface for storing json values in an c++ environment.

The json_variant can hold all necessary json values:

  • json_int
  • double
  • bool
  • std::string
  • json_object
  • json_array
  • json_null_type

A json file can be accurately represented by combining those json_variant values into a tree structure.
